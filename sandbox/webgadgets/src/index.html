The image viewer gadget displays and image from a topic that publishes messages of type sensor_msgs/CompressedImage or sensor_msgs/Image.

Navigation Viewer

The navigation viewer gadget displays the static map, the pose of the robot, and it's planned path. It is possible to set the robot's pose, and assign a goal pose. The map can be panned by clicking the left mouse button and dragging. The map can be zoomed using the middle scroll wheel.
