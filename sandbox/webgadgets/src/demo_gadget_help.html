Purpose:
This gadget is used to view the static map, set/get the robot's pose, and set/get the robot's goal pose.
How To:
  • Set the pose of the robot.
    1. Select the "Set Pose" button
    2. Left-click on the map where the robot is located.
    3. While holding the left mouse button, drag the mouse to orient the robot.
  • Set the goal pose
    1. Select the "Set Goal" button
    2. Left-click on the map where the robot is suppose to go.
    3. While holding the left mouse button, drag the mouse to orient the goal pose.
  • Pan the map
    1. Left-click on the map and drag.
  • Zoom the map
    1. Scroll the middle mouse wheel to zoom or out
